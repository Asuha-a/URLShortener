# INSTALL.md

## clone this repository

```terminal
git clone git@github.com:Asuha-a/URLShortener.git
```

## Set up pre-commit

```terminal
pip install pre-commit
```

pre-commitが入ったことを確認する。

```terminal
pre-commit --version
```

pre-commitをhooksに追加する。

```terminal
pre-commit install
```

### Change the file permission

```terminal
chmod +x make_pb.sh
```
